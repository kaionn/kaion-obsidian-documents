# Obsidian 蓄積ドキュメント

このリポジトリは、Obsidian で管理している知識の蓄積ドキュメントです。

## ディレクトリ構成

- `00_inbox/` - 新しく追加されたノートや未整理の情報
- `01_sources/` - 参照元やソース情報
- `02_working/` - 作業中のドキュメントや進行中のプロジェクト
- `03_output/` - 完成したドキュメントや成果物
- `Clippings/` - Web クリッピングや保存した情報

## 使用方法

1. 新しい情報は `00_inbox/` に追加
2. 整理・分類後、適切なディレクトリに移動
3. 作業中のプロジェクトは `02_working/` で管理
4. 完成したドキュメントは `03_output/` に配置
