# Obsidian 蓄積ドキュメント

このリポジトリは、Obsidian で管理している知識の蓄積ドキュメントです。

## ディレクトリ構成

```
.
├── 00_inbox/          # 新しく追加されたノートや未整理の情報（Git 管理外）
├── 01_sources/        # 参照元やソース情報
│   ├── ai_tools/      # AI ツール関連の記事・メモ
│   ├── iac/           # Infrastructure as Code 関連
│   ├── references/    # 技術参考資料
│   └── tool_obsidian/ # Obsidian の使い方
├── 02_working/        # 作業中のドキュメントや進行中のプロジェクト
├── 03_knowledge/      # 蓄積されたナレッジ
│   └── claude-code/   # Claude Code での開発ログ・学び
├── 04_output/         # 完成したドキュメントや成果物
├── 999_templates/     # ノートのテンプレート
└── Clippings/         # Web クリッピングや保存した情報
```

## ワークフロー

1. 新しい情報は `00_inbox/` に追加（個人的なメモのため Git 管理外）
2. 整理・分類後、`01_sources/` の適切なサブディレクトリに移動
3. 作業中のプロジェクトは `02_working/` で管理
4. 学んだことや開発ログは `03_knowledge/` に蓄積
5. 完成したドキュメントは `04_output/` に配置
